Задайте права доступа на файл `file.txt` 

С помощью цифрового обозначения

## Детали

В начале посмотрите права доступ на файл:

```
ls -l file.txt
```{{execute}}

Теперь поменяем права этому файлу:

```
chmod 764 file.txt
```{{execute}}

Ещё раз посмотрим на права файла:

```
ls -l file.txt
```{{execute}}
