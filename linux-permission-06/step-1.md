Добавьте файлу право на выполнение

И запустите его

## Детали

В начале посмотрите права доступ на файл:

```
ls -l file.txt
```{{execute}}

Попробуем запустить файл:

```
./script.sh
```{{execute}}

Добавим право выполнения:

```
chmod +x file.txt
```{{execute}}

Посмотрим новые права:

```
ls -l file.txt
```{{execute}}

Ещё раз попробуем запустить файл:

```
./script.sh
```{{execute}}
